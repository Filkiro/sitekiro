Se HTML é o corpo do site, CSS é as roupas, CSS cria a estilização do site, ou seja, deixa ele mais bonito. A codificação do CSS não é tão complicada, mas tem bastante coisa para se ver. CSS contém diversos códigos para a estilização, ajuste de resolução de imagem/site e enfim, tem bastante coisa a se ver. Ao contrario do HTML, você não precisa colocar um comando inicial, mas deve colocar um comando dentro do head no HTML, <link rel="stylesheet" href="Diretório de seu CSS">, assim você estará conectando seu HTML ao CSS e poderá criar sua estilização.

O CSS se baseia com as tags, id's e class's, isso porque você necessita identificar para o navegador o que é que você está mexendo e em qual parte que você está mexendo.

EX: body{color: white; font-size: 20px; font-family: "Josefin Sans"; background-color: #161e24;}

No exemplo acima, temos determinado a tag do <body> a cor branca para as letras, afinal o comando é color: white, 20px de tamanho da fonte, afinal o comando é font-size: 20px, o tipo da fonte é Josefin Sans, afinal o comando é font-family: "Josefin Sans" e a cor de fundo é um hexadecimal, que será abordado na lista dos comandos.

A partir deste exemplo é possível ter uma noção do que fazer. A estrutura do CSS depende dessas tags, id's ou class's, sempre que você querer modificar algo do HTML, você deverá indicar ou sua tag, ou sua id, ou sua class.

Exemplo com Tag: h1{...}

Exemplo com Id: #titulo{...}

Exemplo com Class: .titulo{...}

Com os exemplos acima da para ter uma noção sobre a estrutura base, existem uma diferença para determinar a Id e a Class, a Id sempre irá usar "#" no inicio para determinar que é uma Id no CSS, e o Class sempre usará "." no inicio para determinar que é uma class. Um detalhe bom a se colocar é a diferença entre os dois, Id só pode se usar uma vez, quanto a Class pode ser usada varias vezes, por exemplo se eu determinar uma class "texto" para todos os <p>, no CSS você só irá precisar usar o ".texto" somente uma vez para a estilização do <p>.

Vamos para uma tabelinha com alguns comandos básicos:

Código Função
font-size Define o tamanho da fonte
font-family "Familia" da fonte ou o tipo dela
text-decoration Serve para adicionar ou retirar uma decoração do texto
text-align Define a orientação do texto, se ele fica no centro, na esquerda, direita e etc.
color Este comando define a cor da letra
backgroud-color Define a cor de fundo do site
margin Define o tamanho da margem. Utiliza-se 4 números para formar ele, EX: margin: 10px 10px 10px 10px;, cada valor representa a localidade onde a margem irá ficar, o 1° valor é em cima do objeto, o 2° é a direta, o 3° é embaixo e o 4° é a esquerda
padding Mesma estrutura do margin, entretanto, ele faz uma margem dentro do objeto.
border Cria uma borda em volta de seu objeto. Exemplo de Sintaxe: border: 10px solid.
border-radius Cria um arredondamento nos cantos do border;
border-color Define a cor da borda
width Mexe com a Largura
height Mexe com a Altura
left Desloca o objeto para a esquerda. Necessita de um outro comando específico para seu funcionamento
right Desloca o objeto para a esquerda. Necessita de um outro comando específicopara seu funcionamento
top Define o quão para baixo ele vai, se usar números negativos ele irá subir.
position Define o tipo de posicionamento que o objeto deve assumir. Necessário este comando para o top, left e o right funcionarem.
trasition Cria uma transição no seu objeto.
:hover Sua Sintaxe é diferente, ele não vai dentro das chaves, ele vai após sua tag/id/class. EX: h1:hover{...}. Ele significa que quando o ponteiro do mouse estiver em cima do objeto, ele irá ter alguma mudança
@font-face Serve para importar fontes dentro da memória interna, ou pelo link de um site. Ele usa o comando src: url para importar.

A partir daqui teremos comandos mais complexos e com Sintaxes um pouco mais complexas, um exemplo é o @media.
