Primeiramente, tem que se ter em mente um ponto, HTML é o esqueleto do site, ou seja, é o que da a forma, e não vai ser necessariamente bonito, e sim vai ser somente a base de onde você vai começar. No HTML, normalmente se usa o VScode para fazer a "programação", ou seja, tenha o VScode.

No HTML, é utilizado as chamadas tags, que tem determinados significados, varia de tag para tag, mas antes de falar delas, temos que dar uma introdução do HTML, para que o navegador onde estará sendo executado seu site, entenda o que está acontecendo, e o que é o seu código. Para isso, você escreverá na primeira linha, "html", e aparecerá uma lista onde terá "HTML:5", clique na opcão e será gerado um texto automático. No <title>, altere o texto e coloque o que bem entender, isso vai ser seu titulo que aparecerá em cima, onde fica sua aba do site. Dentro do <body> é onde seu código será realmente feito a partir das tags. Existem tags e tags sêmanticas, tags sêmanticas ajudam a organizar seu site de forma melhor, principalmente quando você precisar encontrar determinado comando para fazer uma alteração.

Aqui está uma tabela com as tags:

Código Função
<head> Conteúdo básico do HTML, não aparece no código fonte
<body> Tudo que é vísivel no Código fonte, estrutura onde vai seu HTML
<H1> Titulo do seu Site
<H2 - H6> Sub-Titulo do seu Site
<p> Criar um Paragráfo
<b> Deixar o texto em Negrito
<i> Deixar o texto em Itálico
<ul> Lista desordenada
<ol> Lista ordenada
<li> Define os itens de uma lista
<hr> Cria uma linha de "Separação no texto". Não se fecha </hr>
<mark> Criar o marcador em seu texto
<link> Pegam arquivos de fora de seu código/arquivo
<div> Cria uma Divisão/Caixa do seu conteúdo
<id> Ela é colocada dentro de uma tag, serve para distinguir uma tag de outra tag com o mesmo comando. EX:<p>, <p id="id do p">
<img> Adiciona a imagem que você quer a partir de um Diretório
<table> Cria uma tabela, deve ser usado em conjunto com alguns comandos específicos como: <tr>, <td> e contém comando de estrutura como o: <thead>, <tbody> e o <tfooter>.
<button> Cria um botão
<input> Campo de inserção. É também possível criar botões com ele.

Essas foram as tags mais básicas.

Agora temos as Tags Sêmanticas:

Código Função
<header> Cabeçalho do site
<nav> É sua navegação. EX: Botões que levam a outra parte.
<main> Conteúdo principal do seu site, onde você colocará a maioria das tags
<article> Você colocará informações aqui, normalmente se coloca <section> dentro desta tag
<section> Separações por seções do seu site, coloca-se informações dentro desta tag, como <h1>, <p> e etc para formar seu site.
<article> Você colocará informações aqui, normalmente se coloca <section> dentro desta tag
<details> Cria uma aba da qual pode ser aberta, onde terá as informações que você definiu
<summary> Resumindo, você cria o "titulo" do <details>
<figure> Insere uma Imagem/Figura
<figcaption> Vai dentro do <figure>, e serve para colocar um texto embaixo da imagem
<a> Pegam arquivos de fora de seu código/arquivo
<footer> Rodapé do Site
<em> Deixa o texto em Itálico
<strong> Deixa o texto em Negrito
<cite> É utilizado quando se Cita algo de algum site, é utilizado na fonte de onde você tirou a citação. É comumente utilizado junto ao <q>
<q> Usado na frase da citação, adiciona as aspas.
<time> Usado para definir a data, e alterando seu formato. EX: Ao invés de ser 26/06/2008, você pode mudar isso usando o <time> e deixar 26/6/08.
